# MIT-BIH Arrhythmia Database

## Download Instructions

Download the MIT-BIH Arrhythmia Database from:
**[PhysioNet - MIT-BIH Arrhythmia Database v1.0.0](https://physionet.org/content/mitdb/1.0.0/)**

## Setup

1. Download all the record files (`.atr`, `.dat`, `.hea` files)
2. Place them in this `data/mitbih/` directory
3. You should have files numbered from 100-234 (e.g., `100.atr`, `100.dat`, `100.hea`)

## File Structure

After downloading, your directory should look like:
```
data/mitbih/
├── 100.atr
├── 100.dat
├── 100.hea
├── 101.atr
├── 101.dat
├── 101.hea
...
```
